Turn Arabic customer reviews into actionable business decisions.

Rai'iy - رأيي is an end-to-end NLP pipeline and analytics dashboard built specifically for Arabic customer feedback. It combines fine-tuned sentiment analysis, unsupervised topic modelling, and LLM-powered recommendations to help businesses understand what their customers are saying — and what to do about it.

Features
-  Arabic Sentiment Analysis — Fine-tuned CamelBERT model (trained and tested) classifying reviews as Positive, Negative, or Mixed with confidence scores
-  Automatic Topic Discovery — BERTopic clusters reviews into meaningful themes without manual labeling
-  AI Business Recommendations — We integrated Groq-powered LLaMA 3.3 (LLM) to generate per review insights and strategic overall recommendations in Arabic. 
- Dashboard — Streamlit app with KPI metrics, topic distribution charts, negative topic heatmaps, and a topic explorer
- Arabic Text Cleaning — Custom preprocessing pipeline handling Arabic-specific normalization

To run, open the app2.py file in the folder in vs code, and in a new terminal run: Python -m streamlit run app2.py
Note: Once the code has began running, it will take some time to load so please be patient:)
